# Suffix Arrays

Suffix is a "subsequence" of a string

Suffix Array is the array of all possible subsequences in lexographical order.

"ababba"
- a
- ba
- bba
- abba
- babba
- ababba
- ""

In lexigraphical order
- ""
- a
- ababba
- abba
- ba
- babba
- bba

Written via index:
- 6
- 5
- 0
- 2
- 4
- 1
- 3
